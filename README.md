# Log Analysis Tool

A Python-based GUI application for analyzing log files against reference files.

## Features
- Import reference and log files
- Detect matching signals
- Check occurrences and values
- Display results in a simple GUI
- Built for faster validation workflows

## Technologies Used
- Python
- pandas
- tkinter

## How to Run
1. Install Python 3.x
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python signal_checker_gui.py
